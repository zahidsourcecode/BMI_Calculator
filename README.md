# BMI Calculator App

A Flutter app to calculate BMI, view results on a gauge, and save history locally.

## Built With

| Tool | Version |
|------|---------|
| Flutter | 3.41.7 |
| Dart | 3.11.5 |
| Java | 17 |
| Gradle | 8.14 |
| Android Gradle Plugin | 8.11.1 |
| Kotlin | 2.2.20 |

## Getting Started

```bash
flutter pub get
flutter run
```
